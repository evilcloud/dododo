# Revision history for dododo

## 0.2.0.0 -- 2023-10-11

- New Features:
  - Editable config file: Users can now edit the configuration file directly.
  - Ability to change default editor: Users can now set their preferred editor as the default.
  - Open current tasks and config files with default editor: Users can now open the current tasks file and config file using the default editor.
  - Ability to reset config: Users can now reset the configuration to its default values.
- Improvements:
  - Help: The help text has been improved for better clarity and readability.
- Refactoring:
  - The codebase has been refactored for more efficient operation.

## 0.1.0.0 -- 2023-09-24

- Initial Release:
  - Introduced the foundational version of the microtask system.
  - This release embodies the envisioned concept of what a microtask system could potentially be.
- Implementation Insights:
  - While the current implementation is robust and addresses the core functionalities of the system, its real-world efficacy remains to be evaluated.
- Testing Perspective:
  - The software has undergone rigorous testing to ensure minimal bugs.
  - The primary focus of this release is to gauge the system's societal impact and user reception, rather than just its technical performance.
